**File:** `construction/combo_portfolio/src/domain/combo_test.go`

#### 2a. `ComboName` value-object properties

- [x] **2a.1** Property — *valid name always constructed*: for any string of length 1–100 (generated by `rapid.StringMatching`), `NewComboName` must return a non-error result and `String()` must equal the input.
- [x] **2a.2** Property — *empty string always rejected*: `NewComboName("")` must always return `ErrInvalidComboName`.
- [x] **2a.3** Property — *name over 100 chars always rejected*: for any string of length 101–500, `NewComboName` must return `ErrInvalidComboName`.

#### 2b. `NewCombo` item-count invariant

- [x] **2b.1** Property — *fewer than 2 items always rejected*: for item slices of size 0 or 1, `NewCombo` must return `ErrInvalidItemCount`.
- [x] **2b.2** Property — *more than 10 items always rejected*: for item slices of size 11–50, `NewCombo` must return `ErrInvalidItemCount`.
- [x] **2b.3** Property — *2–10 items always accepted*: for any item slice of size 2–10 with unique `SimpleSku` values, `NewCombo` must return a non-nil `*Combo` with no error.
- [x] **2b.4** Property — *duplicate simpleSku always rejected*: for an item slice containing at least one repeated `SimpleSku`, `NewCombo` must return `ErrDuplicateItem` regardless of item count.

#### 2c. Share / Visibility invariants

- [x] **2c.1** Property — *new combo has no share token*: for any valid combo created via `NewCombo`, `ShareToken()` must be `nil` and `Visibility()` must be `VisibilityPrivate`.
- [x] **2c.2** Property — *Share() sets public + token*: after calling `Share(token)` with any non-empty `ShareToken`, `Visibility()` must equal `VisibilityPublic` and `ShareToken()` must be non-nil and equal to `token`.
- [x] **2c.3** Property — *MakePrivate() atomically revokes token*: for any combo that has been shared, calling `MakePrivate()` must result in `Visibility() == VisibilityPrivate` **and** `ShareToken() == nil` in the same aggregate state — verifying atomicity.
- [x] **2c.4** Property — *MakePrivate is idempotent*: calling `MakePrivate()` twice in sequence must produce the same final state as calling it once.
- [x] **2c.5** Property — *share → make-private → re-share cycle is consistent*: after the full cycle (Share → MakePrivate → Share), combo must be public with a non-nil token.

#### 2d. Rename invariant

- [x] **2d.1** Property — *rename always updates Name()*: for any valid new name, after `Rename(newName)`, `Name()` must equal `newName`.
- [x] **2d.2** Property — *rename does not change item list*: item list, shopper ID, and combo ID must be identical before and after `Rename`.

#### 2e. Domain events

- [x] **2e.1** Property — *NewCombo always emits exactly one ComboCreated event*: `PopEvents()` after creation must return a slice of length 1 containing a `ComboCreated` event.
- [x] **2e.2** Property — *PopEvents is destructive*: calling `PopEvents()` twice returns events on the first call, and an empty (or nil) slice on the second call — for all valid operations.
- [x] **2e.3** Property — *OwnedBy is correct for all shoppers*: `OwnedBy(c.ShopperID())` must always return `true`; for any *different* random shopper ID, it must return `false`.

## Implementation Notes

### PBT Library Choice

Use **`pgregory.net/rapid`** (v1.x):  
```go
import "pgregory.net/rapid"

func TestComboNameValid(t *testing.T) {
    rapid.Check(t, func(t *rapid.T) {
        s := rapid.StringMatching(`[^\x00]{1,100}`).Draw(t, "name")
        _, err := domain.NewComboName(s)
        if err != nil {
            t.Fatalf("expected valid for %q, got %v", s, err)
        }
    })
}
```

### In-Memory Repository for Tests

Extract the `inMemoryComboRepository` from `mockserver/main.go` into a shared test helper at `domain/testhelper_test.go` (package `domain_test`) or duplicate a minimal version in each `_test.go` file. Do not expose it from production packages.

### Stub/Fake Pattern for Application Tests

Define interfaces for the services used by application handlers to enable injection of test doubles:
```go
// In test files only — no production code changes needed
type fakeComboRepo struct { ... }
func (r *fakeComboRepo) Save(ctx, combo) error { ... }
```

`rapid` generators for domain types (reusable across test files):
```go
func genValidItems(t *rapid.T, n int) []domain.ComboItem { /* generate n items with unique SimpleSku */ }
func genValidName(t *rapid.T) domain.ComboName           { /* generate 1-100 char name */ }
func genShopperId(t *rapid.T) domain.ShopperId           { /* non-empty string */ }
```
